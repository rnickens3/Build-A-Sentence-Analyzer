# Build-A-Sentence-Analyzer
Utilizes loops to determine what characters or words are in any sentence.

## Concepts Learned
* Loops and Iteration
* For... of Loop
* For... in Loop
* While Loops
* Do... While Loops
* Break & Continue Statements

## Features
- Counts vowels and consonants separately
- Detects punctuation marks
- Counts words even with extra spaces
- Demonstrates use of loops, if statements, and string methods
